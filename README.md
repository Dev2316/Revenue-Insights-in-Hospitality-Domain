# Revenue Insights in Hospitality Domain

This project focuses on analyzing hospitality revenue data using Power BI to generate actionable business insights. The goal is to help a hotel chain improve its performance, optimize pricing strategies, and regain market share using data-driven decision-making.

## Project Overview

AtliQ Grands, a chain of luxury and business hotels, has been facing a decline in revenue and market share due to increasing competition and poor strategic decisions.

To overcome this, the company decided to adopt **Business Intelligence and Data Analytics**. This project simulates that scenario by analyzing historical booking data and building a dashboard to generate meaningful insights.

## Problem Statement

- Declining revenue and market share
- Lack of data-driven decision-making
- No in-house analytics team
- Need to identify trends and improve business strategies

## Objectives

- Analyze hotel booking and revenue data
- Create key performance metrics (KPIs)
- Build an interactive Power BI dashboard
- Generate insights for business improvement
- Support decision-making using data

## Dataset Description

The dataset consists of multiple tables:

- `dim_date.csv` – Date-related information
- `dim_hotels.csv` – Hotel details
- `dim_rooms.csv` – Room category information
- `fact_bookings.csv` – Booking-level data
- `fact_aggregated_bookings.csv` – Aggregated booking data

These datasets help analyze revenue, occupancy, and booking trends across cities and hotel types.

## Tools and Technologies Used

- Revenue
- Total Bookings
